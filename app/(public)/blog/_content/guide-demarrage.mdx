Ce guide vous aidera à démarrer rapidement avec **OuiConnect** et à tirer le meilleur parti de notre plateforme.

## Étape 1 : Création de votre compte

La première étape consiste à créer votre compte professionnel. C'est simple et rapide :

- Rendez-vous sur la page d'inscription
- Remplissez le formulaire avec vos informations
- Validez votre email
- Et voilà, vous êtes prêt !

## Étape 2 : Configuration de votre profil

Une fois connecté, personnalisez votre profil :

```js
// Exemple de configuration
const profile = {
  nom: "Votre nom",
  métier: "Votre profession",
  disponibilités: ["Lundi", "Mardi", "Mercredi"]
}
```

### Informations importantes

Assurez-vous de remplir toutes les sections de votre profil pour maximiser votre visibilité.

## Étape 3 : Commencer à recevoir des rendez-vous

C'est parti ! Vous pouvez maintenant :

1. Partager votre lien de réservation
2. Configurer vos créneaux horaires
3. Recevoir des notifications pour chaque nouveau rendez-vous
